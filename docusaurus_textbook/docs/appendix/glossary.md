# Glossary

- **ROS 2**: Robot Operating System 2, middleware for robotics.
- **URDF**: Unified Robot Description Format — XML format for robot models.
- **Xacro**: XML macro language for generating URDFs.
- **DDS**: Data Distribution Service — the communication layer used by ROS 2.
- **VLA**: Vision-Language-Action — multimodal robotics architecture.
- **Sim-to-Real**: Transferring models/controllers trained in simulation to physical robots.
- **Jetson**: NVIDIA family of embedded devices for edge AI.
